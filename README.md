Project Goal
My objective was to conduct an investigation within the "Threat Hunting Simulator" to identify and analyze a potential security threat within the company's infrastructure. I focused on validating the hypothesis that an attacker may have leveraged a compromised third-party software package to gain initial system access and establish persistence.

Description of Actions
Throughout this project, I performed the following steps:

Context Assessment: I reviewed the briefing regarding a strange file appearing on the system and examined the Threat Intel report, which indicated a coordinated supply chain attack campaign targeting open-source ecosystems.

Documentation Review: I studied the organizational structure of PawPressMe, including employee details and their respective logged-in hosts, to better understand the environment.

Incident Analysis:

I investigated the alert queue to identify security events.

I analyzed incoming emails containing suspicious external links.

I utilized analysis tools (TryDetectThis) to verify the security of specific URLs.

Response: During the investigation, I confirmed that the firewall successfully blocked an outbound request to a blacklisted external URL.

Conclusion
As a result of my efforts, I successfully identified activity related to phishing attempts and the use of malicious links. I confirmed the effectiveness of the firewall, which blocked an attempt to connect to a malicious resource, thereby preventing the potential progression of the attack. The simulation allowed me to apply my skills in using SIEM systems, analyzing alerts, and reconstructing the attack chain.


Objetivo do Projeto
Meu objetivo foi conduzir uma investigação dentro do "Threat Hunting Simulator" para identificar e analisar uma potencial ameaça à segurança na infraestrutura da empresa. Foquei em validar a hipótese de que um atacante poderia ter utilizado um pacote de software de terceiros comprometido para obter acesso inicial ao sistema e estabelecer persistência.

Descrição das Ações
Ao longo deste projeto, realizei as seguintes etapas:

Avaliação de Contexto: Analisei o briefing sobre um arquivo estranho que apareceu no sistema e examinei o relatório de Inteligência de Ameaças, que indicava uma campanha coordenada de ataque à cadeia de suprimentos voltada para ecossistemas de código aberto.

Revisão de Documentação: Estudei a estrutura organizacional da PawPressMe, incluindo detalhes dos funcionários e seus respectivos hosts logados, para entender melhor o ambiente.

Análise de Incidente:

Investiguei a fila de alertas para identificar eventos de segurança.

Analisei e-mails recebidos que continham links externos suspeitos.

Utilizei ferramentas de análise (TryDetectThis) para verificar a segurança de URLs específicas.

Resposta: Durante a investigação, confirmei que o firewall bloqueou com sucesso uma solicitação de saída para uma URL externa na lista negra.

Conclusão
Como resultado dos meus esforços, identifiquei com sucesso atividades relacionadas a tentativas de phishing e ao uso de links maliciosos. Confirmei a eficácia do firewall, que bloqueou uma tentativa de conexão com um recurso malicioso, evitando assim a possível progressão do ataque. A simulação me permitiu aplicar minhas habilidades no uso de sistemas SIEM, análise de alertas e reconstrução da cadeia de ataque.
